WordPress Freelancer FAQ

Apr 2nd, 2013

<div class="title">
  <h2>'WordPress Freelancer Profile Widget' shows a profile information about someone else (not mine)?</h2>
</div>
<div class="content">
  <p>After activating the widget for the first time, the default values are used to generate the widget front end output. You need to enter your freelancer.com username into the username field in the plugin settings page.</p>
</div>

<div class="title">
  <h2>'WordPress Freelancer Widgets' show only the plain information without styling. Why widget styles are missing ?</h2>
</div>
<div class="content">
  <p>You need to have wp_head() function in your theme template. This plugin uses wp_head() to output enqueued styles. Put wp_head(); in the page HTML head section.</p>
</div>

<div class="title">
  <h2>Where are the 'WordPress Freelancer Wigets' ? Appearance > Widgets ?</h2>
</div>
<div class="content">
  <p>You need to have at least one sidebar in your theme. Otherwise there will be no Appearance > Widgets section in the WordPress Admin. Register a new sidebar in your 'functions.php' and add it to your theme template.</p>
</div>

<div class="title">
  <h2>I don't like the default appearance of the widgets. How do I change the styling of widgets ? </h2>
</div>
<div class="content">
  <p>Go to WordPress Freelancer Plugin Dashboard in the WordPress Admin. At the bottom of the Profile Tab, you can find the link to the Customizer. You can change the colors, borders, etc. and see the effects of the changes you made in the preview panel. After you are done with tweaking the widget appearance click 'Save'.</p>

The other option is to use a filter hook described in ‘WordPress Freelancer Hooks API’.

</div>

<div class="title">
  <h2>No output seems to be generated in 'WordPress Freelancer Affiliate Widget' ?</h2>
</div>
<div class="content">
  <p>Affiliate widget uses javascript to obtain and display affiliate project listing. This plugin needs 'wp_footer()' function in your theme functions file. WordPress Freelancer Affiliate Widget javascripts enqueue in the footer through 'wp_footer()'. Put wp_footer(); in your theme template footer.</p>
</div>

<div class="title">
  <h2>Can I see the raw (xml or json) api query responses ?</h2>
</div>
<div class="content">
  <p>Yes you can view the raw responses that are returned by freelancer.com api upon querying for information. Those responses are stored in WordPress transients. WordPress Freelancer Plugin allows you to extend the plugin itself using WordPress Freelancer Hooks API.</p>

Whenever you hook into the WordPress Freelancer Hooks API you are given these api response object for manipulation. You can view all of them in one place.

Go to WordPress Freelancer Plugin Dashboard and then to Settings Tab. In the tabbed panel you’ll find the information about raw api responses.

</div>

<div class="title">
  <h2>I need to reset the 'WordPress Freelancer Plugin' to the initial state. How do I do that?</h2>
</div>
<div class="content">
  <p>Under each of the settings tabs, there is a 'Reset Defaults' button which you can use to reset the settings of that tab to defaults.</p>

Data retrieved by querying the freelancer.com api is stored in WordPress transients. If you need to clear all transients ( There are only two transients at a given time if you didn’t change the username ) go to settings tab in the WordPress Freelancer Plugin Dashboard and check the settings field for ‘Clear transients on Save’. Upon saving all transients will be deleted.

</div>
